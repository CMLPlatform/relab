# RELab Docs

Standalone Astro Starlight documentation app for the RELab platform.

## Quick Start

```bash
fnm use 24 || nvm use 24
just install
just dev
```

The docs site runs on <http://127.0.0.1:8012>.

## Common Commands

```bash
just build
just check
just test-e2e
just format
just audit
```
